# 문재의 핵심
- BFS
- 경우에따라 bfs를 다르게 나눌 수 있는가?

# 알고리즘
- 모든 maze의 row, col을 순회하면서
    - maze가 R이거나 G경우
    - maze의 셀이 적록색맹이 아직 방문하지 않은경우
        - bfs를 통해 적록색맹의 영역을 구함
        - 적록색맹은 visted=1을 통해 저장함
    - maze의 셀이 적록색맹이 방문했지만 일반 사람이 방문하지 않은경우
        - bfs를 통해 영역을 구함
        - visited=2를 통해 적록 색맹 저장
- 영역반환

# 자료구조 
- maze: 2차원 배열
- visited: 2차원 배열
