# 2. 알고리즘
### 1. 해결방법
```
- 2차원 배열을 선언
    - 배열의 원소: 현 노드가 루트일때 최소 얼리어답터
    - 노드가 0일때: 얼리어답터가 아님 
        - 인접노드들이 반드시 얼리어답터들이어야함
        - 따라서 자식노드들 중 얼리어답터만 부모노드에 더해줌
    - 노드가 1일때: 얼리어답터임
        - 인접노드들이 반드시 얼리어답어일 필요는 없음
        - 따라서 자식노드들 중 얼리어답터의 최소값을 더해줌
```
